RaidouOS: A 64-bit multitasking graphical operating system

This is my little OS project for x86-based processors with long mode support. It has its own little bootloader and functions in kernel mode. It is theoretically turing complete, as one can write and execute brainfuck programs. It uses PS/2 to interface with the keyboard and mouse, and renders video using VESA standards. It can play audio through the PC speaker. It has a desktop with a window management system with a process system managed by a round robin scheduler.

Plans for next release (gamma): better multitasking (DONE), a text editor (TODO).
